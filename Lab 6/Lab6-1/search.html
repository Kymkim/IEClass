Search Engines

Dogpile Returns the top seatch results from multiple other search engines.
Ask Allows common questions to be entered as a search key. Provides a smart search for products and weather.
Google Provides general keyword and phrase searching. Provides definition search by typing "definition" and a term.
HotBot Provides general keyword search, also the toolbar allows you to search your local computer. (No longer a seach engine. The site is now a VPN Service)
Bing General keyword search, also allows searches to be limited to specific file types.
Yahoo Provides news search and tabbed categories to help locate information.
